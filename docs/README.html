race-clock

This is a program that is intended to be used in a race. There are two different programs meant to be used together.

Manual

Prerequisites

  • Java 17

File explanation

Filename Description
race.properties Settings for the program, for example where the files are stored
README.md This file, which includes instructions for how to use the program
register-v2.0.jar The GUI used to register start and end times
result-v2.0.jar The program used to generate the result file, intended to be run in a terminal
release-info.txt Contains information about the current release (patch notes)
gui_files/drivers.txt Information about which start number corresponds to which driver
gui_files/start.txt After using the GUI, this file will be updated with information from it. The GUI also loads this information on startup.
gui_files/end.txt After using the GUI, this file will be updated with information from it. The GUI also loads this information on startup.
acceptance_tests This folder contains all our passed acceptance tests

Options

Option-name Description
driversPath Defines the path for driver-names
startTimesPath Defines the path for start times
endTimePath Defines the path for end times
driverResultPath Defines the path where the result file is written
raceName Defines the name of the race
resultFormat Defines the type of wished sortingmethod

How to use

  1. Start the GUI by running register-v2.0.jar
  2. Use the buttons Start and Finish to register drivers start and finish times
  3. Exit the GUI and open a terminal in the project folder
  4. Run the command java -jar result-v2.0.jar to generate the result file
  5. Open the result file to see the results
